# Bland AI Integration

This guide shows how to integrate Maven with Bland AI to collect payments.

## Overview

<Steps>
  <Step title="Create Maven session">
    Your server creates a payment session via Maven API
  </Step>
  <Step title="Transfer via pathway">
    Use Bland's transfer node to send customer to Maven
  </Step>
  <Step title="Handle webhook">
    Maven notifies you when payment completes
  </Step>
</Steps>

## Pathway Setup

Add a transfer node in your Bland pathway:

1. Create a **Custom Tool** node that calls your server to create a Maven session
2. Add a **Transfer Call** node that transfers to the Maven phone number
3. Configure your webhook endpoint to receive payment results

## Server Endpoint

```javascript
// POST /bland/collect-payment
app.post('/bland/collect-payment', async (req, res) => {
  const { amount, from } = req.body;

  // Create Maven session
  const session = await fetch('https://api.trymaven.com/v1/sessions', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${process.env.MAVEN_API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      project: 'my-project',
      caller: from,
      amount: amount * 100,
      currency: 'USD',
      mode: 'charge',
      callback: process.env.BLAND_PHONE_NUMBER
    })
  });

  const { phone_number } = await session.json();

  return res.json({
    transfer_to: phone_number
  });
});
```
