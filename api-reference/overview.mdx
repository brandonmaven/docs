## Base URL

```
https://api.trymaven.com/v1
```

## Authentication

All requests require an API key in the `Authorization` header:

```bash
Authorization: Bearer sk_live_your_api_key
```

See [Authentication](/get-started/authentication) for details.

## Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/sessions` | [Create a payment session](/api-reference/sessions/create) |
| `GET` | `/sessions/:id` | [Get session details](/api-reference/sessions/get) |
| `GET` | `/sessions?caller={phone}` | [Get session by caller](/api-reference/sessions/get-by-caller) |
| `POST` | `/sessions/:id/cancel` | [Cancel a session](/api-reference/sessions/cancel) |
| `GET` | `/projects` | [List all projects](/api-reference/projects/list) |
| `GET` | `/auth/check` | [Verify API key](/api-reference/auth/check) |

## Error Handling

Errors return an appropriate HTTP status code and a JSON body:

```json
{
  "detail": "Missing required field: caller"
}
```

| Status Code | Description |
|-------------|-------------|
| `400` | Bad request - invalid parameters |
| `401` | Unauthorized - invalid API key |
| `404` | Not found - resource doesn't exist |
| `429` | Rate limited - too many requests |
| `500` | Server error |
