---
title: "Create Tools"
api: "POST /voice/api/v1/vapi/create-tools"
description: "Auto-create Maven payment tools in your Vapi account"
---

Automatically create Maven payment tools in your Vapi account. This creates two server tools that your Vapi assistant can use to collect payments:

- **SendContextToMaven** - Calls set-context to create a payment session and get the transfer destination.
- **GetContextFromMaven** - Calls get-context to retrieve the payment result after transfer-back.

Your Maven API key is embedded in the created tools so Vapi can authenticate when calling Maven endpoints.

## Request Body

<ParamField body="vapi_api_key" type="string" required>
  Your Vapi API key. Used to create tools in your Vapi account.
</ParamField>

<ParamField body="assistant_id" type="string">
  Vapi assistant ID to attach the created tools to. If not provided, tools are created but not attached to any assistant.
</ParamField>

<ParamField body="project" type="string">
  Default project ID or slug to embed in the created tools. If not set, the Vapi assistant must supply `project` at call time.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether both tools were created successfully.
</ResponseField>

<ResponseField name="tools_created" type="array">
  List of tool names that were created (e.g., `["SendContextToMaven", "GetContextFromMaven"]`).
</ResponseField>

<ResponseField name="error" type="string">
  Error message if tool creation failed.
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.trymaven.com/voice/api/v1/vapi/create-tools \
  -H "Authorization: Bearer mvn_live_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "vapi_api_key": "vapi-abc123-def456",
    "assistant_id": "asst_xyz789",
    "project": "my-app"
  }'
```

```javascript Node.js
const response = await fetch('https://api.trymaven.com/voice/api/v1/vapi/create-tools', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer mvn_live_your_api_key',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    vapi_api_key: 'vapi-abc123-def456',
    assistant_id: 'asst_xyz789',
    project: 'my-app'
  })
});

const result = await response.json();
console.log(result.tools_created); // ["SendContextToMaven", "GetContextFromMaven"]
```

```python Python
import requests

response = requests.post(
    'https://api.trymaven.com/voice/api/v1/vapi/create-tools',
    headers={'Authorization': 'Bearer mvn_live_your_api_key'},
    json={
        'vapi_api_key': 'vapi-abc123-def456',
        'assistant_id': 'asst_xyz789',
        'project': 'my-app'
    }
)

result = response.json()
print(result['tools_created'])  # ["SendContextToMaven", "GetContextFromMaven"]
```
