---
title: "Get Session"
api: "GET /v1/sessions/{id}"
description: "Retrieve details for a specific session"
---

Retrieve details for a specific session, including payment status, processor details, and error information.

## Path Parameters

<ParamField path="id" type="string" required>
  Session UUID
</ParamField>

## Response

<ResponseField name="session_id" type="string">
  Unique session identifier (UUID)
</ResponseField>

<ResponseField name="project_slug" type="string">
  Project slug
</ResponseField>

<ResponseField name="caller" type="string">
  Customer's phone number in E.164 format
</ResponseField>

<ResponseField name="status" type="string">
  Current session status (see table below)
</ResponseField>

<ResponseField name="amount" type="number">
  Amount in cents
</ResponseField>

<ResponseField name="currency" type="string">
  Currency code (e.g., `USD`)
</ResponseField>

<ResponseField name="email" type="string">
  Customer's email address, if provided
</ResponseField>

<ResponseField name="full_name" type="string">
  Customer's full name, if provided
</ResponseField>

<ResponseField name="customer_id" type="string">
  Merchant's customer identifier, if provided
</ResponseField>

<ResponseField name="processor" type="object">
  Payment processor details. Present after payment is processed.
  <Expandable title="processor properties">
    <ResponseField name="stripe_payment_intent_id" type="string">
      Stripe PaymentIntent ID (charge mode)
    </ResponseField>
    <ResponseField name="stripe_charge_id" type="string">
      Stripe Charge ID (charge mode)
    </ResponseField>
    <ResponseField name="payment_method_id" type="string">
      Stripe PaymentMethod ID (tokenize mode)
    </ResponseField>
    <ResponseField name="card_brand" type="string">
      Card brand (e.g., `visa`, `mastercard`)
    </ResponseField>
    <ResponseField name="card_last4" type="string">
      Last 4 digits of the card
    </ResponseField>
    <ResponseField name="receipt_url" type="string">
      Stripe-hosted receipt URL (charge mode)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="error" type="object">
  Error details when status is `payment-failed`.
  <Expandable title="error properties">
    <ResponseField name="code" type="string">
      Machine-readable error code
    </ResponseField>
    <ResponseField name="message" type="string">
      Human-readable error message
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="mode" type="string">
  Payment mode: `charge` or `tokenize`.
</ResponseField>

<ResponseField name="description" type="string">
  Charge description, if provided.
</ResponseField>

<ResponseField name="callback" type="string">
  Callback phone number, if provided.
</ResponseField>

<ResponseField name="memory" type="string">
  Context shared with the receiving agent. `null` if not provided.
</ResponseField>

<ResponseField name="metadata" type="object">
  Custom key-value data.
</ResponseField>

<ResponseField name="completed_at" type="string">
  ISO 8601 timestamp of when the session reached a terminal state. `null` if still in progress.
</ResponseField>

## Session Status Values

| Status | Description |
|--------|-------------|
| `created` | Session created, waiting for call |
| `collecting` | Customer connected, collecting card info |
| `processing` | Processing payment |
| `payment-success` | Payment completed successfully |
| `payment-failed` | Payment failed |
| `expired` | Session expired (no call received) |
| `cancelled` | Session was cancelled |

<RequestExample>

```bash cURL
curl https://api.trymaven.com/v1/sessions/{session_id} \
  -H "Authorization: Bearer {api_key}"
```

```javascript Node.js
const response = await fetch(
  'https://api.trymaven.com/v1/sessions/{session_id}',
  { headers: { 'Authorization': 'Bearer {api_key}' } }
);

const session = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.trymaven.com/v1/sessions/{session_id}',
    headers={'Authorization': 'Bearer {api_key}'}
)

session = response.json()
```
