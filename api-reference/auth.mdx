# Auth

Use the auth endpoint to verify that your API key is valid and working.

## Check Auth

<ParamField path="GET" method="/v1/auth/check" />

Verify your API key and return account information.

### Response

```json
{
  "authenticated": true,
  "org_id": "org_xxx",
  "mode": "test",
  "message": "API key is valid. Mode: test"
}
```

<ResponseField name="authenticated" type="boolean">
  Always `true` for a valid key. Invalid keys return a `401` error instead.
</ResponseField>

<ResponseField name="org_id" type="string">
  Your organization ID
</ResponseField>

<ResponseField name="mode" type="string">
  API key mode: `test` or `live`
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable status message
</ResponseField>

### Example

<CodeGroup>

```bash cURL
curl https://api.trymaven.com/v1/auth/check \
  -H "Authorization: Bearer mvn_test_xxx"
```

```javascript Node.js
const response = await fetch('https://api.trymaven.com/v1/auth/check', {
  headers: {
    'Authorization': 'Bearer mvn_test_xxx'
  }
});

const auth = await response.json();
console.log(auth.mode); // "test" or "live"
```

```python Python
import requests

response = requests.get(
    'https://api.trymaven.com/v1/auth/check',
    headers={'Authorization': 'Bearer mvn_test_xxx'}
)

auth = response.json()
print(auth['mode'])  # "test" or "live"
```
